We are the **Gencare IT Department**, building **secure, scalable, and innovative healthcare technology solutions**.

Our work powers real hospital platforms — combining clean engineering with a mission to make healthcare **accessible, efficient, and patient-friendly**.

- 🔭 Currently working on: **Shri Ram Multi-Speciality Hospital Web Platform**
- 🌱 Focus areas: Hospital Management Systems, Patient Portals, Appointment & Emergency Systems
